# jokenpo
Jogo de jokenpô em Java.

## Descrição

Implementação do clássico jogo de pedra, papel e tesoura (jokenpô) desenvolvido em Java.

## Como Jogar

O jogador escolhe entre:
- **Pedra** (Rock)
- **Papel** (Paper)
- **Tesoura** (Scissors)

O computador faz sua escolha e o resultado é determinado pelas regras clássicas do jogo.

## Requisitos

- Java 8 ou superior

## Como Executar

```bash
javac jokenpo.java
java jokenpo
```

## Estrutura do Projeto

- `App.java` - Classe principal com a lógica do jogo

## Regras

- Pedra vence Tesoura
- Tesoura vence Papel
- Papel vence Pedra
- Escolhas iguais resultam em empate
